  • Introduction

  • Introduction

    EVM , Electronic Voting Machine is developed for digitizing the conduct of school parliamentary elections in school. It is an open source software written in Python language. As it follows GNU Public License anyone is free to make changes to the code subject to the clauses of the GPL.

  • How to Begin

  • 1: Select Class and Division

    On loading the software the first window opening is the control panel where all settings are to be done. First of all you have to select Class(es) and Division(s) in which election is to be conducted and make necessary data inputs

  • 2: Add Candidates

    Next step is adding Candidate: Use the textbox to enter the names of the candidate. Click on "Add Candidate" button to save the candidate. Maximum 10 candidates are allowed. As it supports unicode,You can type in English, Malayalam and other languages.

  • 3: Remove Candidates

    In order to remove a candidate select the class and division first. From the listed names of candidates select the name to be removed. Click on the button Remove

  • 4: Start Poll

    Click on 'Start Poll' to start polling. It opens a new window with the names of all the candidates. The green signal indicates it is ready for voting. The voters[students] can use mouse to vote for the candidate of their choice. After each vote the candidate keys are disabled and a red signal indicating "not ready" will appear. The polling officer[the teacher] can use the 'space key' to enable voting again. Then green signal will appear. As the voters are permitted to use mouse only, they will not be able to close, minimize or interrupt the poll in any way.The teacher may sit behind the screen in such a way that the confidentiality of the election process is well respected

  • 5: Returning to the Control Panel

    Press the hot key Ctrl+E to go back to the control panel window once the election process is over. The control panel gives you access to a handful of features like result and continue poll

  • 6: Publish Result

    After the poll the result can report can be generated either by selecting individual classes or Navigating to the menu item File->Consolidated Result. If you dont select any class, the result button on the control panel itself will give the consolidated result

  • Frequently Askeds

  • Start Poll & Continue Poll: What is the Difference?

    Start poll function is used to begin a fresh election for a class. It will erase all polled votes for the partcular class. On the other hand, Continue poll is helpful to resume poll from where it stopped. This might prove useful in case of power failure or unexcpected closure. When continue poll is selected all the votes cast so far would be retained

  • Will Remove Candidate/ Add Candidates Options erase all data?

    Not in deed. The Remove candidates/Add candidates options will delete all polled votes of the candidates of the selected class. However, the votes of other classes will remain totally unaffected

  • Can I use same machine to conduct Election to Different Classes

    Certainly. It is designed keeping this in mind. You can conduct elections in various classes one after the other. The data of the conducted polls would be safely saved. And the result may be taken and declared after the completion of the whole election process

  • What is Consolidated Report

    When the whole election process is completed the whole result can be genereated in a single click, rather than manually clicking each class. Use the Menu File-> Consolidated Report for the purpose
