Getting Started

About

Smart School application is part of an ambitious initiative to computerise most of the school activities especially. This application helps manage examination by providing progress reports, promotion list and a few other reports and even send sms and email

Downloading

The software project is hosted by github.com and it can be downloaded from here

Installation

The software, though written in Python, which is a cross platform language, is developed keeping in mind Linux users. As the windows compatibility is not tested the installation steps are provided for Ubuntu distribution only. There are a number of methods to install a software on Ubuntu. Two easy ways are explained here
    1. Extract the downloaded compressed file
    2. Right click the smart_school_all.deb file and open with Ubuntu Software Centre or GDebi Packge Installer
      3. .
    3. Give administrative password if prompted
    4. Click on Install Package or Install
    5. Done
    2. Or
    1. Right Click on install.sh, take properties->Permission and check 'Allow executing as a program'
    2. Open the Terminal by Pressing Ctr+ Alt + T
    3. Navigate to the folder where the extracted folder resides, using command 'cd'
    4. press enter after typing the command ./install.sh
    5. Done
After the installation the software can be found in Application-> Education-> Smartschool

How to

Most of the menues and options are self explanatory and intutive. However, here is the step by step manual for making things clearer and easier for you

Login

The software is password protected in order to safeguard important data and avoid misuse. Two inbuilt users are available for login
  1. admin
  2. teacher

Admin user is the most privilleged user who can make potential changes to the data and settings. So, it is advised to limit the access to this account to one or two users who can judiciously manage the software without causing data loss.

In order to enter exam score, generate progress report, promtion list etc, you only require the previllege of a 'teacher' user. However,only the admin user has the necessary previllege to manage divisions, edit students' details, etc,

default passwords

  • admin:-admin
  • teacher:-teacher

Institution Settings

Once successfully logged in as admin, the Admin Dash Board gets open from which "Institutio" button can be selected. Fill the text boxes and click Save Button

Academic Year and Division

Select "Year and Division"" from Admin Dash, divisions for any academic year can be added or removed using this window. To add divisions from A-k, just type A-K in the division text box instead of adding one by one manually

The link 'Add Next Academic Year' automatically creates next academic year.

Set CE and TE

Before entering examination scores, you have to set the CE and TE marks for each standards for the particular academic year. Use CE TE Settings option for this

Working Days

Total workigs days for three terms is required for processing promotion list and progress report of that term. Working Days button in the Dash borsd will help do it.

Add, Edit or Remove Students

The software comes with a very powerful tool that helps add the details of all students in the school in a single click. Import from Sampoorna feature is a fast, reliable and safe way to add all students data to Smart School Software. As the software does not make any changes to the database of Sampoorna and does not save the user name or password it is completely safe. It is definitely a great idea to use this to initialize the database. Afte importing, you may have to edit the roll numbers once, which can easily be done by logging in as teacher user. In subsequent years, it might be better to import the data of 8th standard only after promoting other students using 'promote' option

Manage Students tool is meant for adding, editing, removing individual students details. It is available in Admin Dash

Backup Data

It is always a good idea to backup vital data on machine. 'Export Database' tool is designed for that. It generates a file with .xam extension. If you encounter any data loss this backed-up file can be imported using 'Import Database', which restores the data.

Reports

Progress Report

Progress report of individual classes can be had by logging in as 'teacher'. Select Performance Card from Reports Menu on the top of the window.

Promotion List

Promotion list can also be generated using the Report menu option found in the individual classes when logged in as 'teacher' The promotions lists of multiple classes may be made using Promotion List tool in Admin Dash

Custom Report

Custom report is designed keeping in mind the need for academic scores of students in schools for various purpose. This feature enables you to create reports of one or more classes with columns and data of your choice. Like Name, CE of One Subject, TE of the Subject, Total Mark, Grade, etc. The result is given in the form of spreadsheet file so that it can be used for editing and modificatios

Data Import

There is a feature made available for performing data sync and transport across computers. In a school, it is not possible to install the software on a single computer and do the work on it always. This feature allows us to work on different computers and keep our work safe. A teacher can enter the mark of a class pertially in one computer and the other part in another machine. In such cases, this tool is helpful in syncing the data. In order to sync data Export Database first and take the backup file ___.xam in a storage device. Login as admin in the targer computer and import the backup file using "Import Database". Importing will not affect normally the other classes or data already saved in the target machine. However,it is safer to back the database of the targer computer also before import.
